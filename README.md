# Plataforma do NEJ POA
O objetivo é centralizar os dados das EJ's num lugar de fácil acesso para todos.

## Funcionalidades
1. Adicionar pessoas na EJ
2. Enviar contratos
3. Estipular metas
4. Área de LogIn
5. Dados da Rede
